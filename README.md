# RetreatHut 🌴

RetreatHut è un'applicazione web che permette agli utenti di scoprire e prenotare spazi di coworking e coliving in diverse località nel mondo.

Gli utenti possono creare un account, effettuare il login, esplorare le location disponibili, salvare i propri luoghi preferiti, effettuare prenotazioni e gestire i propri soggiorni.

## ✨ Funzionalità

- Registrazione e login degli utenti
- Esplorazione di location coworking e coliving
- Aggiunta e rimozione delle location dai preferiti
- Creazione e cancellazione delle prenotazioni

## 🛠️ Tecnologie utilizzate

**Frontend**

- React
- React Router
- React Bootstrap
- Vite

**Backend**

- Node.js
- Express.js
- MongoDB
- Mongoose
- Autenticazione JWT
